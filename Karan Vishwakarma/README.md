### Run Locally

To run **Karann** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/karanvishwakarmaa/C-Personal-Portfolio
```

Windows:

```bash
git clone https://github.com/karanvishwakarmaa/C-Personal-Portfolio
```
